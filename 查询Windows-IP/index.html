查询Windows IP

Windows IP
运维 Windows
发布日期:   2023-04-01

方式一：

一、如何知道我们本机的IP地址

第一步：首先电脑上面的快捷键win + R

第二步：然后在打开这个选项中输入cmd,然后回车

第三步：输入命令：ipconfig,然后进行回车

注意这时候我标注出来的IPv4的地址就是一般所需要的地址

方式二：

第一步：搜索控制面板，然后进行回车

第二步：选择网络和Internet

第三步：点击网络共享中心

​ 第四步：点击更改适配器->双击以太网->点开详细信息就可以看到IP地址

以上就是如何查看电脑IP的方式，当然查看的IP的方式不仅仅是这这两种，也可以在设置网络里面进行直接的查看，还有其他方式等等…
