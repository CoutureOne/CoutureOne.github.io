1.修改默认远程端口号

1.1 关闭SELINUX

如果不关闭SELINUX修改远程端口号无法生效(天坑之一)

[root@localhost ~]# vim /etc/sysconfig/selinux
# SELINUX=enforcing 修改为
SELINUX=disabled
[root@localhost ~]# reboot

[root@localhost ~]# setstatus

1.2 修改防火墙

[root@localhost ~]# firewall-cmd --zone=public --add-port=11111/tcp --permanent
success
[root@localhost ~]# firewall-cmd --reload
success
[root@localhost ~]# more /etc/firewalld/zones/public.xml
<?xml version="1.0" encoding="utf-8"?>
<zone>
  <short>Public</short>
  <description>For use in public areas. You do not trust the other computers on 
  networks to not harm your computer. Only selected incoming connections are accepted.</description>
  <port port="11111" protocol="tcp"/>
  <forward/>
</zone>

1.3 修改sshd的配置

[root@localhost ~]# cd /etc/ssh/
[root@localhost ssh]# vi sshd_config

1.4 修改端口port

#Port 22
Port 31234

1.5 配置文件的有效性和密钥的完整性测试

[root@localhost ~]# sshd -t
# 没有问题的情况下就是显示
[root@localhost ~]#

1.6 重启sshd服务

[root@localhost ~]# systemctl restart sshd.service

1.7 测试

[root@localhost ~]# ss -lntp
State      Recv-Q    Send-Q   Local Address:Port     Peer Address:Port    Process
LISTEN     0         128      0.0.0.0:31234          0.0.0.0:*            users:(("sshd",pid=11111,fd=3))

 如果出现上面的结果说默认远程端口号明修改已经成功完成

2.网卡固定IP地址

2.1 找到网卡信息并且修改配置文件

[root@localhost ~]# cd /etc/sysconfig/network-scripts/
[root@localhost ~]# ls
[root@localhost ~]# ifcfg-ens192
[root@localhost ~]# vi ifcfg-ens192

BOOTPROTO=none
ONBOOT=yes
IPADDR=你的静态IP地址
NETMASK=你的子网掩码
GATEWAY=你的默认网关
DNS1=首选DNS服务器
DNS2=备用DNS服务器

DEVICE=eth0
NAME=eth0
BOOTPROTO=none
ONBOOT=yes
IPADDR=192.168.1.100
NETMASK=255.255.255.0
GATEWAY=192.168.1.1
DNS1=8.8.8.8
DNS2=8.8.4.4

2.2 重启网路服务命令

sudo systemctl restart NetworkManager

2.3 重启整个系统

sudo reboot

测试正常访问或者是ping通网络，说明修改成功

3.安装VMware Tools

3.1 挂载VMware Tools

在安装VMware Tools之前，确保系统中已安装所需的构建工具和依赖项。在Rocky Linux中，可以使用以下命令安装所需软件包：

sudo dnf install gcc kernel-devel kernel-headers make perl elfutils-libelf-devel
  • 在Rocky Linux中，打开终端并创建一个用于挂载的目录：
sudo mkdir /mnt/cdrom
  • 然后将VMware Tools安装介质挂载到该目录：
sudo mount /dev/cdrom /mnt/cdrom

3.2 将VMware Tools安装文件解压到临时目录中：

tar -zxvf /mnt/cdrom/VMwareTools-*.tar.gz -C /tmp/

3.3 运行安装程序：

cd /tmp/vmware-tools-distrib/
sudo ./vmware-install.pl

3.4 重启系统

sudo reboot
